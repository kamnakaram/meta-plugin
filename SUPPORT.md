# Support

## Getting Help

| Need | Where to go |
|------|-------------|
| **Bug report** | [Open a GitHub Issue](https://github.com/AI-Marketing-Hub/meta-ads/issues/new/choose) |
| **Feature request or question** | [GitHub Discussions](https://github.com/AI-Marketing-Hub/meta-ads/discussions) |
| **Security vulnerability** | [GitHub Security Advisory](https://github.com/AI-Marketing-Hub/meta-ads/security/advisories/new) (private) |
| **Contributing** | See [CONTRIBUTING.md](CONTRIBUTING.md) |

## Before Opening an Issue

1. Check the [README](README.md); installation, usage, and FAQ cover the most common questions
2. Search [existing issues](https://github.com/AI-Marketing-Hub/meta-ads/issues) to avoid duplicates
3. Include your OS, Python version, the command that failed, and the full error output

## Response Time

Issues and discussions are monitored regularly. Most questions receive a response
within a few days. Security reports are acknowledged within 48 hours; see
[SECURITY.md](SECURITY.md) for details.
